# P Language Grammar

This repo implements a tree-sitter grammar for the P language.

## NOTE

This grammar is entirely AI generated using Claude Opus 4.6. I haven't really proofread any
of the code. Use this grammar at your own risk. 

Remarkably, Claude was able to generate and validate this grammar in under an 
hour using the P language source code as reference. In my day to day use of this 
grammar in neovim, I've found it to be generally excellent. In fact, in many ways better 
quality for syntax highlighting than the official P extension in vscode.
